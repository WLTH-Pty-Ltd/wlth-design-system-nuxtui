# @wlth/design-system

Shared Nuxt layer for all WLTH products. Provides components, composables, tokens, and fonts.

## Usage

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  extends: ['@wlth/design-system'],
  modules: ['@nuxt/ui'],
})
```

## Documentation

- **[UX Patterns & UI Configuration](docs/UX_PATTERNS.md)** — North star reference for layout architecture, design tokens, component conventions, and patterns. All WLTH products should follow this guide.

## Playground

```bash
npm install
npm run dev
```
